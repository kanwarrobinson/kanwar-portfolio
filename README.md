# Kanwar Robinson - AI Engineer Portfolio

A modern, interactive portfolio website showcasing my work as an AI Engineer at AppViewX.

## Features

- 🎨 Modern, animated UI with Framer Motion
- 💻 Interactive terminal component displaying developer info
- 🌓 Dark/Light theme toggle
- 📱 Fully responsive design
- ⚡ Fast and optimized with Vite
- 🎯 Smooth scrolling and animations
- 📧 Contact form with backend integration

## Tech Stack

### Frontend
- React 18
- Vite
- Framer Motion (animations)
- React Icons
- React Type Animation

### Backend
- Node.js
- Express
- Nodemailer

## Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open browser at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Run Backend Server

```bash
npm run server
```
