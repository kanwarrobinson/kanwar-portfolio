## ✏️ Quick Customizations:

### Change Terminal Content:
Edit `src/components/Terminal.jsx` (lines 14-26)

### Add More Projects:
Edit `src/components/Projects.jsx` - add to projects array

### Update Colors:
Edit `src/styles/index.css` - modify CSS variables

### Add Your Photo:
Place in `public/` folder and update component paths

## 🌐 Deploy Your Portfolio:
